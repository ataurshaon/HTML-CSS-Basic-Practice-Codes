--Class 20-- 

1. CSS Position- 4 parameters in position- Position Static, Position fixed, position absolute, position relative.

2. When we use position relative or absolute by default the div class have 4 parameter such as top, bottom, left, right. 

3. When we use position relative this will move from it's own place to top, bottom, left, right.

4. When use position absolute the div class will move on the full window of web page.

5. If we want to move one div to another then it will show one into another. If we want to move the seconde div around the first div then we should use position relative in first div and position absolute in second div. Then second div will move around by first div.

We can move the class of the div tag by using inspect element of the parameter of the position relative or absolute.

 As an example

.div1{
    width: 400px;
    height: 200px;
    background: red;
    position: relative;    
}

.div2{
    width: 100px;
    height: 100px;
    background: green;
    position: absolute; 
}

6. For moving to center of anything use as below

top: 50%;
left: 50%;
transform: translate(-50%, -50%);

Full css code as below:

div1{
    width: 400px;
    height: 200px;
    background: red;
    position: relative;   
}
.div2{
    width: 100px;
    height: 100px;
    background: green;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);   
}
